7up Sprite Editor
(c) 2008,2009 kerm1t (Dr.Pepper)


Edit makespindizzy+start.bat, replace your WinVice path to allow compile and
execute your sprite drawing.


Controls
--------
[F1..F4]        - Select Pencil color
[0..9 u. A..F]  - Set current pencil's color
<Space>         - draw
<Cursor Keys>   - guess what?
