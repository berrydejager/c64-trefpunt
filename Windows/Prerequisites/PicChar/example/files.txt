Explanation of the files in the example-directory;


alien.bmp	- Original BMP file
alien.prg	- C64 Hires version of the BMP file
alien.chr	- Charset generated by PicChar using alien.prg
alien.sc1	- Screendata generated by PicChar usign alien.prg
example.asm	- Example source code how to show a converted picture
example.prg	- Binary of the example source code. Run it with a SYS4096 in an emu or on real C64
files.txt	- This file
